# 🛠️ 바닐라 자바스크립트로 Chrom app 만들기

<br />

### 📌 구현 기능

- 초단위 시계
- 로컬스토리지를 이용한 로그인(로그인 후 투두 리스트 작성 가능)
- 로컬스토리지를 이용한 투두 리스트(등록, 삭제)
- 랜덤 배경화면 및 랜덤 명언
